# Machine Learning Examples

## Install requirements in venv
```
    # Linux
    python -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt

    # Windows
    python -m venv venv
    venv\Scripts\activate
    pip install -r requirements.txt

```
